# 设计规范文档

## 1. 设计概述

### 1.1 设计目标
- 保持与 Dify 现有设计系统一致
- 提供清晰直观的资源级别选择体验
- 展示专业的系统监控数据可视化

### 1.2 设计原则
- **一致性**：复用 Dify 组件和设计语言
- **清晰性**：资源级别差异一目了然
- **反馈性**：操作结果即时反馈
- **专业性**：监控数据专业展示

## 7. 交互规范

### 7.1 表单交互

#### 输入验证
- 实时验证：输入后 500ms 延迟验证
- 错误提示：输入框下方红色文字
- 成功提示：输入框右侧 ✓ 图标

#### 提交反馈
- 加载状态：按钮显示 Spinner
- 成功：Toast 提示，自动关闭
- 错误：Toast 提示，手动关闭

### 7.2 列表交互

#### 排序
- 点击表头排序
- 支持升序/降序切换
- 默认排序：创建时间倒序

#### 筛选
- 顶部筛选栏
- 支持多条件组合
- 筛选结果实时更新

#### 分页
- 底部页码导航
- 每页 20 条（可选 10/50/100）
- 显示总条数

### 7.3 模态框交互

#### 创建/编辑
- 标题：创建资源配置 / 编辑资源配置
- 内容：表单
- 操作：[取消] [确认]

#### 确认删除
- 标题：确认删除
- 内容：删除后不可恢复，是否继续？
- 操作：[取消] [删除]

#### 测试连接
- 标题：测试连接
- 内容：显示测试进度和结果
- 操作：[关闭]

### 12.2 组件示例

```tsx
// ResourceLevelCard.tsx
import { Card, CardHeader, CardTitle, CardContent } from '@/components/ui/card'
import { Badge } from '@/components/ui/badge'
import { Button } from '@/components/ui/button'

interface ResourceLevelCardProps {
  level: 'low' | 'medium' | 'high'
  title: string
  description: string
  features: string[]
  metrics: {
    latency: string
    qps: string
  }
  selected?: boolean
  onSelect: () => void
}

export function ResourceLevelCard({
  level,
  title,
  description,
  features,
  metrics,
  selected,
  onSelect,
}: ResourceLevelCardProps) {
  const levelColors = {
    low: 'border-green-500 bg-green-50',
    medium: 'border-blue-500 bg-blue-50',
    high: 'border-purple-500 bg-purple-50',
  }

  return (
    <Card
      className={cn(
        'cursor-pointer transition-all duration-200',
        'hover:shadow-lg',
        selected && levelColors[level]
      )}
      onClick={onSelect}
    >
      <CardHeader>
        <div className="flex items-center justify-between">
          <CardTitle className="text-lg">{title}</CardTitle>
          <Badge variant={level}>{level}</Badge>
        </div>
        <p className="text-sm text-muted-foreground">{description}</p>
      </CardHeader>
      <CardContent>
        <ul className="space-y-2">
          {features.map((feature) => (
            <li key={feature} className="text-sm">{feature}</li>
          ))}
        </ul>
        <div className="mt-4 pt-4 border-t">
          <div className="flex justify-between text-sm">
            <span>延迟：{metrics.latency}</span>
            <span>QPS：{metrics.qps}</span>
          </div>
        </div>
        <Button className="w-full mt-4" variant={selected ? 'default' : 'outline'}>
          {selected ? '已选择' : '选择此方案'}
        </Button>
      </CardContent>
    </Card>
  )
}
```
